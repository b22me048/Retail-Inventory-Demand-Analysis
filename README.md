## Project Overview

This project focuses on analyzing retail inventory data to understand stock movement, product demand, and inventory trends. The goal is to identify fast-moving and slow-moving products and observe patterns in stock availability.

## Dataset

* Contains product-level inventory and sales records
* Includes features like product category, stock levels, sales quantity, and timestamps

## Tools & Technologies

* Python (Pandas, Matplotlib)
* Excel

## Key Analysis

* Data cleaning and preprocessing (handling missing values and duplicates)
* Time-based demand analysis
* Identification of fast-moving vs slow-moving products
* Analysis of stock-outs and overstock trends

## Results & Insights

* Observed variation in product demand across time
* Identified categories with consistent vs irregular demand
* Highlighted patterns in stock availability

## Conclusion

The analysis provides insights into inventory behavior and helps in understanding how demand patterns can affect stock management.

## How to Run

1. Install required libraries:

   ```
   pip install -r requirements.txt
   ```
2. Run the notebook:

   ```

   ```
